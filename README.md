# AI/ML Engineering Internship Tasks - DevelopersHub Corporation

This repository contains the completed tasks for the AI/ML Engineering Internship.



## Tasks Overview

### Task 1: Exploring and Visualizing a Simple Dataset
* **Objective:** Load, inspect, and explore dataset features using statistical parameters and charts.
* **Dataset:** Iris Dataset (Built-in Seaborn)
* **Key Findings:** Identified clear cluster separations for the `setosa` species via scatter plots and detected minor outliers in `sepal_width` using box plots.

### Task 2: Predict Future Stock Prices (Short-Term)
* **Objective:** Build a time-series regression model to predict the next day's closing stock price.
* **Dataset:** Apple Inc. (AAPL) via `yfinance`
* **Models Applied:** Linear Regression
* **Key Results:** Achieved an exceptional fit matching tracking trends tightly with a low Mean Absolute Error (MAE) of 0.98.

### Task 3: Heart Disease Prediction
* **Objective:** Train a binary classification model to evaluate the risk metrics of cardiac conditions.
* **Dataset:** Heart Disease UCI Dataset
* **Models Applied:** Logistic Regression
* **Key Results:** Reached an accuracy of 88.33% with an outstanding ROC-AUC score of 0.95.
